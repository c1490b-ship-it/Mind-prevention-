# Mind-to-Mind Prevention Set

This repository contains config and templates to reduce accidental secret leaks,
prevent social-engineering attack vectors, and enforce safer contribution flow.

Files included:
- `.github/workflows/` — CI for secret scanning & dependency updates
- `.pre-commit-config.yaml` — local pre-commit checks (secrets, formatting)
- `SECURITY.md` — how to report security issues
- `.github/ISSUE_TEMPLATE/security.md` — security report template
- `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `CODEOWNERS`

Follow the "Setup" section below to enable protections.
